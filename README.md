# 🖨️ Gitesh Enterprises – Printing Business Website

A modern, professional, and fully responsive animated business website built for **Gitesh Enterprises**, designed to showcase printing services and enable customers to place orders easily via WhatsApp.

---

## 🚀 Project Overview

This project is a **dynamic and visually engaging website** for a printing company that provides multiple services like flex printing, LED boards, ID cards, and more.

The goal of this website is to:

* Attract local customers
* Showcase services professionally
* Provide an easy ordering system via WhatsApp
* Deliver a premium UI/UX experience

---

## 🎯 Features

### 🎨 Modern UI/UX Design

* Clean **White + Black professional theme**
* Smooth animations and transitions
* Hover effects and micro-interactions
* Gradient overlays and glow effects
* Fully responsive (Mobile, Tablet, Desktop)

---

### 🏠 Hero Section

* Background visuals of printing machines & boards
* Animated business name and tagline
* Dynamic changing tagline:

  * All Printing Solutions in One Place
  * Flex, Boards, ID Cards & More
  * High Quality Printing at Affordable Prices
  * Custom Designs for Every Need
* CTA Button: **Order Now**

### 🛠️ Services Section

* Flex Printing
